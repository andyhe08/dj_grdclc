The grade calculator at grdclc.com is developed and maintained solely by Andy He, a computer science student at Duke University in the class of 2024. This personal project aims to help me develop the skills of HTML, CSS, JavaScript, Python, Django web framework, SQL databases, and deployment to Heroku as well as offering a easy-to-use, simple, and efficient grade calculator within a class for high school and college students. Feel free to use the developer console, accessed by F12 or Ctrl+Shift+C to view the JavaScript algorithm.

Signing in with a profile allows students to store all of their classes in one place, as a robust database storage system ensures data security. To learn more about the specifics about the application, and specifics of data storage, click here.

Method:

The calculation is performed through the following steps:

  1. Within a category, sum all of your points and divide by the sum of the total points (to have all assignments be of equal value, use normalize function)
  2. Find the category's contribution by multiplying this average by the weight (be sure to enter the weight as a percent, not decimal)
  3. Repeat for all categories
  4. Sum the contributions for the final score

Buttons:

Adds a category under current category with one default assignment

Deletes the current category if there is more than one category

Converts all grades within current category to be out of 100 points

Adds a assignment under current assignment

Deletes current assignment

Converts assignment's grade to be out of 100 points

Saves sheet as an interactive excel file, with file name being the class name

Saves sheet to user profile using database management feature (must be logged in)
